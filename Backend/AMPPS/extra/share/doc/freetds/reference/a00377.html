Detailed Description

Functions to manipulate DBDATETIME structures.

Defined by Sybase only. These are not implemented:

  • dbdate4cmp()
  • dbdate4zero()
  • dbdatechar()
  • dbdatename()
  • dbdateorder()
  • dbdatepart()
  • dbdatezero()
  • dbdayname()
